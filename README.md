# Sistema de Gestión Integral de Torneos Interescolares (SGITI 2.0)

# Descripción.

SGITI 2.0 es una plataforma destinada a la gestión y organización de campeonatos interescolares.

Permitirá registrar instituciones educativas, equipos deportivos, participantes, entrenadores, árbitros, partidos, resultados y tablas de posiciones.

El objetivo es digitalizar la administración de torneos escolares y facilitar el acceso a la información en tiempo real.

# Tecnologías.

* SQL Server
* Git
* GitHub
* Miro

# Distribución de Responsabilidades.

* Product Owner: Definición de requisitos y prioridades del proyecto.
* Scrum Master: Coordinación del equipo y seguimiento de tareas.
* Diseñador UI/UX: Diseño de la experiencia de usuario e interfaz gráfica.
* Backend: Desarrollo de la lógica de negocio y funcionalidades.
* Base de Datos: Diseño e implementación de la estructura de datos.
* Tester: Verificación y validación de funcionalidades.
* Documentación: Elaboración de documentación técnica y funcional.
* Soporte: Asistencia técnica y apoyo al equipo.
